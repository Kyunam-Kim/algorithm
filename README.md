### * 알고리즘 정의와 분석

```
알고리즘 => 주어진 조건에서 효율적으로 문제를 해결하는 방법

O 표기법 => 알고리즘에서 최악의 성능에 대한 측정 방법

알고리즘에 따른 O 표기법
 - 반복문은 최대 반복 횟수로 계산한다.
 - 중첩된 반복문은 각각의 중첩문의 최대 반복 횟수를 곱셈하여 계산한다.
 - 반복문이 떨어져서 2개 이상 있는 경우는 그 중 가장 큰 값으로 계산한다.
 - if-else문은 알고리즘 성능에 영향을 미치지 않는다.
 - 재귀 호출을 풀어서 계산한다.
```
